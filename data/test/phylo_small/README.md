# phylo_small — core-genome phylogeny e2e fixture

Prokka annotations + a Roary pan-genome for a full end-to-end test of
the `phylogeny` recipe (single-copy core → MAFFT × N → IQ-TREE).

| Path | What |
|---|---|
| `gffs/g{1..4}.gff` | Prokka GFF for 4 phiX174-derived strains |
| `gffs/g{1..4}.ffn` | matching nucleotide CDS (FASTA) |
| `gene_presence_absence.csv` | Roary GPA across the 4 strains |

Four strains (phiX + three 25-SNP variants) give IQ-TREE the ≥4 taxa it
needs for an unrooted ML tree with ultrafast bootstrap, while keeping
the run to a few seconds (~6 single-copy core genes, ~5 kb supermatrix).

## Regeneration

Generated by running the `pangenome` recipe (Prokka × 4 → Roary) on four
phiX variants and harvesting each strain's `.gff` / `.ffn` plus the
Roary `gene_presence_absence.csv` — guarantees the GFF, CDS, and GPA are
mutually consistent (hand-crafting them is error-prone).
