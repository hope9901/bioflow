# methyl_small — WGBS methylation e2e fixture

A tiny synthetic bisulfite dataset for a full end-to-end test of the
`methylation_wgbs` recipe (TrimGalore → Bismark → methylKit).

| File | What |
|---|---|
| `genome.fa` | phiX174 reference (5,386 bp) |
| `sample01_R{1,2}.fastq.gz` | 3,000 directional paired-end bisulfite reads (80 bp) |

The reads are generated by in-silico bisulfite conversion of phiX
fragments: unmethylated C's become T, while CpG C's stay methylated ~70 %
of the time, so methylKit recovers a real, coherent CpG methylation level
(~70 %).  Reads are **directional** — R2 is the reverse complement of the
*same* converted strand R1 came from (the G→A pattern Bismark expects),
half from the top strand and half from the bottom — which is why all
3,000 pairs map at 100 % efficiency.

The genome is **not** committed pre-prepared: the recipe's `bismark_prep`
stage runs `bismark_genome_preparation` at run time, so no
version-tied bowtie2 index lands in git.

## Regeneration

`python scripts/gen_methyl_fixture.py` (deterministic, `seed=42`).

## Used by

`tests/integration/test_full_pipeline_e2e.py::test_methylation_wgbs_full_chain`.
